A self-hosted photo management system that automatically organizes personal photo libraries by people and events using face recognition.
Implements face detection, identity clustering, reference-image visual search, and automatic event grouping based on timestamps.
Built and maintained by Anirudh Dabas using pretrained open-source models, with no custom model training.
